# Smart Contract Grabber (智能合約自動抓取器)
## Used For (用途)
 用於爬取網站上的智能合約，目前支援：
- etherscan

## Environment (運行環境)
- Python3

## Install (安裝方式)
- MAC, Linux
```console
$ . pip-install.sh
```
OR
```shell
$ poetry install
$ poetry shell
```

## Run (執行方式)
```console
$ python main.py
```

## User Interactive (使用者互動的輸入資訊)
- domain (爬取的網站)
- address (合約地址)
- saved directory (爬到合約要儲存的資料夾路徑)
